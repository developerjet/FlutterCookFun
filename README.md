### 主要功能
- 首页食材分类、分类列表
- 选择食材配菜、烹饪、做菜步骤、视频播放
- 菜谱列表、菜谱详情
- 白天夜间模式切换
- 多语言设置

### 技术实现
- Getx实现项目基本脚手架（路由实现、界面传值处理）
- Getx实现控制器数据绑定、传值 
- Getx白天夜间换肤处理
- Getx配置多语言
- getwidget基本控使用
- 列表上下拉加载数据处理
